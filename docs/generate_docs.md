# Інструкція з генерації документації

Для проєкту використовується інструмент **pdoc** (легковагий генератор документації для Python, який не потребує складних конфігурацій).

## Кроки для генерації:

1. Переконайтеся, що ви знаходитесь у віртуальному середовищі проєкту.
2. Встановіть інструмент (якщо ще не встановлено):
    `pip install pdoc`
3. Виконайте команду для генерації HTML-документації у папку `docs/api`:
    `pdoc main.py -o docs/api`
    *(де `main.py` - головний файл з кодом застосунку)*

Після цього оновлена документація буде автоматично доступна на GitHub Pages після пушу в репозиторій.
